Para testar:
1. Extraia o ZIP.
2. Abra o CMD dentro da pasta copfisio-main.
3. Rode: python -m http.server
4. Acesse: http://localhost:8000

Se aparecer codigo HTML puro na tela, voce provavelmente abriu o arquivo errado, abriu o codigo-fonte, ou o navegador carregou cache antigo. Use http://localhost:8000 e pressione Ctrl+F5.

Configuração de Email Automático

Este site é HTML, CSS e JavaScript. Por segurança, ele não usa senha SMTP/Gmail no navegador. Configure abaixo os dados do EmailJS para enviar o comprovante automaticamente após o agendamento.

No template do EmailJS, use {{to_email}} como destinatário e {{reply_to}} como resposta. Variáveis disponíveis: patient_name, appointment_date, appointment_period, appointment_time, patient_doc, patient_email, patient_phone, complaint, message.
